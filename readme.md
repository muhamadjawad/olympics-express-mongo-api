# MONGO DB COMMANDS FRO DATABASES

 ### View all dbs
```
show dbs
```

### create new db
```
use db_name
```

### View current dbs
```
db
```

### Delete DB
go inside db and then 
```
dp.dropDatabase()
```

# MONGO DB COMMANDS FOR COLLECTIONS

### Show Collections 
```
show collections
```

### create collections
```
dp.createCollection("collection_name")
```
